# Update Memory

When a task reveals durable information:

1. Decide whether it belongs in a project manifest or the configured LLM Wiki path.
2. Use manifests for routing metadata: tags, notes, wiki links, dependencies.
3. Use wiki notes for procedures, runbooks, decisions, and debugging recipes.
4. Keep entries short and searchable.
5. Never write secrets.
